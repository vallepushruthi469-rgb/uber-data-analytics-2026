Analyze Uber trip data to identify peak hours, ride patterns, and demand trends using Python.

## Objectives
- Data Cleaning
- Data Analysis
- Peak Hour Detection
- Data Visualization

## Technologies Used
- Python
- Pandas
- Matplotlib

## Dataset
1000 Uber trip records

## Results
- Total Trips: 1000
- Peak Hour Identified
- Hourly and Daily Trip Analysis
